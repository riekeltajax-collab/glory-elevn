Ga naar "Producten"

2

Tik op "Bewerk"

Zoek het shirt, tik op de Bewerk knop. Het formulier klapt open.

3

Kies nieuwe foto's

Tik op het foto-upload vlak → kies foto's uit je galerij. De oude foto's worden vervangen.

4

Sla op en publiceer

Tik "Opslaan" → ga naar 💾 Publiceer → genereer → kopieer → plak in GitHub → commit.

⚠️ Belangrijk over foto's
Foto's worden opgeslagen als link Als je een foto kiest uit je galerij, wordt die omgezet naar een tijdelijke link. Deze werkt alleen zolang je de admin pagina open hebt.

Voor permanente foto's op je site: Upload de foto's ook naar je GitHub images/ map. Gebruik dan bij "Foto URL" de GitHub link: images/jouw-foto.jpg
🔐 Wachtwoord veranderen

Open admin.html op GitHub → zoek naar const ADMIN_PASSWORD → verander de waarde → commit.
