💾 Exporteer & Publiceer
Genereer de code en zet hem live op GitHub in 4 stappen.
⚡ Hoe werkt publiceren? Je beheert producten hier in de admin. Als je klaar bent, genereer je de code en plak je die in producten.js op GitHub. Dat is het enige bestand dat je hoeft aan te passen — de rest van de site blijft ongewijzigd.
Stap 1 — Genereer je productencode
Stap 2, 3 & 4 — Publiceer naar GitHub
2

Open GitHub op je telefoon of computer

Ga naar jouw repository → tap op het bestand producten.js

3

Vervang de inhoud

Tap op het potlood ✏️ → selecteer alle tekst → plak de gekopieerde code → tap "Commit changes"

4

Klaar! 🎉

Je site is binnen 1–2 minuten bijgewerkt. Ververs de pagina om het te zien.

📸 Foto's uploaden via je telefoon
Je hoeft foto's niet te hernoemen — GitHub bewaart gewoon de originele naam (zoals IMG_1234.JPG).
💡 Tip: installeer de GitHub app Download de GitHub app gratis in de App Store of Play Store. Log in met je account. Dan kun je rechtstreeks vanuit je foto's uploaden.
📱 Uploaden via GitHub app (telefoon)
1

Open de GitHub app

Ga naar jouw repository (glory-elevn)

2

Navigeer naar de images/ map

Tap op de map images. Bestaat die nog niet? Tap rechtsboven op + → "Create file" → typ images/leeg.txt → commit. Dan bestaat de map.

3

Upload je foto's

Tap rechtsboven op + → "Upload files" → kies foto's uit je album. Je kunt meerdere tegelijk selecteren. Tap "Commit changes".

4

Kopieer de bestandsnaam

Na upload zie je de foto staan in de map. De naam is exact zoals je hem nodig hebt in het admin formulier — bijv. images/IMG_1234.JPG

💻 Uploaden via computer (browser)
1

Ga naar github.com → jouw repository

Open de images/ map

2

Klik "Add file" → "Upload files"

Sleep meerdere foto's tegelijk vanuit je Finder of Verkenner naar het uploadvenster.

3

Commit changes → klaar

Foto's zijn nu beschikbaar als images/jouw-foto.jpg

📖 Bestaand product wijzigen
Zo pas je snel de naam, prijs, maten of foto's van een bestaand product aan.
Optie A — Via dit beheer scherm (makkelijkst)
1

Ga naar "Alle producten"

Klik in de sidebar op 📋 Alle producten.

2

Klik op "Bewerk"

Het formulier klapt open onder het product. Pas aan wat je wilt.

3

Klik "Opslaan"

De wijziging staat nu in de lijst.

4

Exporteer & publiceer

Ga naar 💾 Exporteer → genereer code → kopieer → plak in producten.js op GitHub → commit.

Optie B — Direct in producten.js (voor als je snel iets wil fixen)
1

Open producten.js op GitHub

Ga naar github.com → jouw repository → tap op producten.js

2

Tap op het potlood ✏️

Nu kun je de tekst direct aanpassen. Zoek het product op naam (bijv. "Ajax").

3

Pas de waarde aan

Bijv. prijs van '€45' naar '€40' — of voeg een speler toe aan de lijst.
Let op: tekst altijd tussen aanhalingstekens '...' houden.

4

Tap "Commit changes"

Binnen 1-2 minuten staat de wijziging live op je site.
