🔑 GitHub Token
Hoe maak je een token aan? 1. Ga naar github.com → Inloggen
2. Profielfoto → Settings → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Note: "Glory Elevn Admin" · Expiration: No expiration
6. Vink ✅ repo aan → Generate token
7. Kopieer de token (begint met ghp_) en plak hem hieronder
⚠️ Bewaar dit token veilig. Hij wordt alleen lokaal opgeslagen op jouw apparaat.
