💡 Hoe werkt de migratie? 1. De admin haalt elke Shopify foto op via een proxy
2. Slaat hem op in de images/ map op jouw GitHub
3. Koppelt automatisch de nieuwe link aan het juiste product
4. Na voltooiing: druk op Publiceren → Shopify kan weg!
⏱️ Hoelang duurt het? Met ~200 foto's duurt dit 5-15 minuten. De admin werkt op de achtergrond — laat je scherm aan staan. Als een foto al bestaat op GitHub wordt die overgeslagen.
